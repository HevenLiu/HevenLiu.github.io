Configuration Of Hexo And NexT

Posted on 2017-08-19 | In 如何使用HexoAndNexT | | Visitors:

背景图添加方式

  • cd /blog所在目录/

  • 进入目录的/themes/next/source/css/_common/components/header/ 目录

  • 打开herder.style文件,将第一句herder的background的值修改为URL的值

    eg1: .header { background: url(‘输入文件所在的URL地址’); }

    eg2: .header { background: url(‘输入文件的本地地址[相对路径]或[绝对路径]’); }

头像添加方式

  • cd /blog所在目录/
  • 进入目录的 /themes/next/ 目录
  • 打开_config.yml文件,搜索关键字avatar (Line:129) , 后面的值修改为URL地址
